
以下以 翻译 人狼转生 为例。


1. 作业-->新书籍.

代码输入  n1576cu

名称输入  人狼转生

网页输入 http://ncode.syosetu.com/n1576cu/

按[新增]按钮。

提示成功后 关闭窗口。


注：如果书籍是在 ncode.syosetu.com 的
填写了网页地址， 后续能够执行 批量新增章节的操作。
不填写的话，就只能手动，一个章节一个章节进行添加。



2. 作业-->新章节

这里是手动添加一个章节的操作。

请确认 书籍 的下拉列表那里， 是 人狼转生

章节代码输入    n1576cu/001
章节名称输入    交易都市リューンハイト攻略戦（前編）

下面的空白区域， 
手动打开浏览器， 访问 
http://ncode.syosetu.com/n1576cu/1/

然后把 正文部分，复制粘贴过来。

按 [保存] 按钮。

提示成功后 关闭窗口。



2.1.  作业-->批量新增章节


请确认 书籍 的下拉列表那里， 是 人狼转生

按 [加载网站] 按钮。

因为 第一话已经手动添加了。 这里 在下拉列表中， 选择 2，3，4，5 话。

按[保存]按钮。


提示成功后 关闭窗口。




 
3. 作业-->机翻

请确认 
书籍 的下拉列表那里， 是 人狼转生
章节 的下拉列表那里，是 n1576cu/001 交易都市リューンハイト攻略戦（前編）


手动打开浏览器， 访问 
http://www.excite.co.jp/world/chinese/


在画面的 ウェブページ翻訳 的位置， 粘贴地址
http://ncode.syosetu.com/n1576cu/1/

然后
把 默认的 中→日  修改为  日→中
把 默认的 訳文のみ表示  修改为  訳文と原文を表示

然后按 右边那个  翻译按钮。

将弹出新窗口，等待翻译结束。 复制翻译结果， 粘贴到程序中。



按 [开始处理] 按钮。

提示成功后 关闭窗口。




注意: 上面的方式, 是通过 excite 网站,  进行机器翻译.
如果需要使用 百度翻译, 进行机器翻译的话, 参考下面的操作步骤.

打开页面
http://fanyi.baidu.com/transpage?query=http%3A%2F%2Fncode.syosetu.com%2Fn1576cu%2F1%2F&from=auto&to=zh&source=url&render=1

目标网页地址,填写到指定的位置, 按翻译按钮。
复制翻译后的结果， 粘贴到程序中。


按 [开始处理] 按钮。

提示成功后 关闭窗口。






4. 作业-->人工翻译

请确认 
书籍 的下拉列表那里， 是 人狼转生
章节 的下拉列表那里，是 n1576cu/001 交易都市リューンハイト攻略戦（前編）

下面是逐行翻译的操作。

主要修改  结果文本的 区域。
修改完毕后， 按 [完成并保存] 按钮，将保存当前修改行。
如果修改了一小部分，需要去别的行查看一下，那么 按 [暂存] 按钮。 保存到数据库。

注: [完成并保存] 的操作 ， 相当于  下拉列表中选择 [已完成]，然后按  [暂存] 按钮的操作。






前面的窗口，选择 隐藏空白行后，前几行， 会遇到名词。

源文本：俺の今の名は、ヴァイトという。皆はあまり気にしていないが、個人的には発音に気をつけて欲しい。
机翻文本：我的现在的名，称为vaito。全部不太介意,不过希望，个人地当心发音。
结果文本：我的现在的名，称为vaito。全部不太介意,不过希望，个人地当心发音。

现在需要创建一个  ヴァイト - 瓦伊特 的名词。


5. 作业-->名称

名词输入：   ヴァイト
机翻文字：vaito
结果文字：瓦伊特
译文说明：瓦伊特 （主角名称）

按 [新增] 按钮后，关闭窗口，  回到 人工翻译 窗口。


人工翻译 窗口,  按 [重新加载名词] 按钮。
然后按[下一行]、[上一行]。  
在 参考用名词区域， 可以看到 
ヴァイト  :  瓦伊特 （主角名称） 

按 [尝试替换名词] 按钮。

源文本：俺の今の名は、ヴァイトという。皆はあまり気にしていないが、個人的には発音に気をつけて欲しい。
机翻文本：我的现在的名，称为vaito。全部不太介意,不过希望，个人地当心发音。
结果文本：我的现在的名，称为 瓦伊特 。全部不太介意,不过希望，个人地当心发音。



--------------------------------------------------------------------------

翻译完毕后。

11. 结果-->生成结果文本

选择好 书 和 章节，  按 [Build] 按钮。

缺点是结果那里， 要鼠标右键，  [全选] 之后， 才能复制出来。


12. 结果-->生成文本文件

选择好 书 和 章节， 以及文件夹，  按 [导出] 按钮。




--------------------------------------------------------------------------

多台电脑之间切换工作。

21.系统-->数据交换

选择好 书 和 章节，  做 导出/导入 的操作。

一般主要是导出 命名 那一章。

还有就是，如果某一章翻了一半， 要去另外一台计算机上翻的话， 就是 这台机器上导出， 那台机器上导入。
